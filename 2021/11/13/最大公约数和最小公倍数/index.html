最大公约数和最小公倍数

输入两个正整数m和n，求其最大公约数和最小公倍数。

输入

两个整数

输出

最大公约数，最小公倍数

样例输入复制

5 7

样例输出复制

1 35

思路:

首先我们需要了解，最小公因数和最小公倍数的相关性质

两数的乘积 / 最大公因数 = 最小公倍数

所以要求最大公因数和最小公倍数，只需求出其一就可直接利用性质，完成题目。

辗转相除法是古希腊求两个正整数的最大公约数的，也叫欧几里德算法，其方法是用较大的数除以较小的数，上面较小的除数和得出的余数构成新的一对数，继续做上面的除法，直到出现能够整除的两个数，其中较小的数（即除数）就是最大公约数。以求288和123的最大公约数为例，操作如下：

288 % 123 == 42

123 % 42 == 39

42 % 39 == 3

39 % 3 == 0

此时3 就是288和 133的最大公约数

结构大致如此 , num1 num2 使用大数取模小数

t = m% n

m = n

n = t

代码如下

#include <iostream>
using namespace std;
int main(){
    int n,m,num1,num2;
    //输入两个数
    int t = 1;
    cin >> num1 >> num2;
    m = num1;
    n = num2;
    if(m>n){
        //进入循环 辗转相除
        while(t != 1){
            t = m % n;
            m = n;
            n = t;
        }
        //求最小公倍数
        int b = (num1*num2) / m; 
        cout << m << " " << b <<endl;
    }else if(m < n){
        while(t!=0){
            t = n % m;
            n = m;
            m = t;
        }
        int b = (num1*num2)/n;
        cout << n << " " << b;
    }else if(m==n){
        //直接输出最小公倍数 公约数都相同
        cout << n << " " << n <<endl;
    }
    
    return 0;

}
