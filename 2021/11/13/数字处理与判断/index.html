数字的处理与判断

题目描述

给出一个不多于5位的整数，要求 1、求出它是几位数 2、分别输出每一位数字 3、按逆序输出各位数字，例如原数为321,应输出123

输入

一个不大于5位的数字

输出

三行 第一行 位数 第二行 用空格分开的每个数字，注意最后一个数字后没有空格 第三行 按逆序输出这个数

样例输入复制

12345

样例输出复制

5
1 2 3 4 5
54321

题解：

分析：这题如果按照常规思路比较麻烦，1.确定循环位数 2.循环正序输出 3. 循环逆序输出 所以我们采用字符串的

解题思路：

将输入的数字按照字符串来存储，用strlen()得到字符串的长度 ，那么第一问就解决了

然后将字符串从左到右输出 再出右往作输出 即可

#include<iostream>
#include <cstring>
//注意事项：
//
//题目说数字位数不超过5，所以定义字符数组长度应不小于6
//
//因为字符串结尾还有结尾符 '\0' 占一位
using namespace std;
int main(){
    char str[10];//	定义字符串长度为10
    cin >> str;
    int len = strlen(str);//求出字符串的长度
    cout << len << endl;
    //从左到右遍历输出

    for(int i = 0; i < len; i++){
        //对于格式的处理
        if(i==0){
            cout << str[i];
        }else{
                cout << " " << str[i];
        }
    }
    cout << endl;
    //逆序输出 中间没有空格
    for(int i = len-1 ; i >= 0; i--){
        cout << str[i];
    }
    cout << endl;
    return 0;
}
